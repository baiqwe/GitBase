# Next 8 High-Value Pages

This document lists the next eight pages worth building after the current homepage, category pages, and existing intent pages are stable. The goal is to expand coverage without weakening the current SEO structure.

## Page selection rules

- Prefer pages with a clear user job, not vague keyword stuffing.
- Prefer pages that can reuse the current object pool with only light curation.
- Prefer pages that can be linked naturally from the homepage, categories, and existing intent pages.
- Keep the page structure consistent:
  - Hero + generator
  - What this page is for
  - How to use it
  - Why this page is different
  - Example objects
  - Related internal links
  - FAQ
  - Trust / curation block

## Internal link policy

- The homepage should feature at least four of these pages in its editorial links block.
- Each new page should link to:
  - one related category page
  - two sibling intent pages
  - one broader parent page such as `/random-object-generator` or `/random-object-to-draw`
- Relevant category pages should link back to the most natural intent pages:
  - `animals` -> drawing, kids, pictionary
  - `household` -> classroom, office, scavenger hunt
  - `food` -> pictionary, ESL, kids
  - `nature` -> drawing, storytelling
  - `funny` -> charades, improv, party games

## Content guardrails

- Do not clone homepage copy and swap nouns.
- Each page should have its own examples, FAQ, and explanation of why the object pool is narrowed that way.
- Keep the generator useful first; the content should support intent, not bury the tool.
- Reuse the same SEO skeleton, but write the explanatory copy independently for every page.
